Bike Sales Dashboard

About this project
I built this Excel dashboard to analyze bike sales data and understand how different factors influence customer purchases.

The project walks through the full process — from raw data to insights — using Excel tools like pivot tables and slicers.
 What I did
- Cleaned and explored the raw dataset
- Created pivot tables to summarize key metrics
- Built an interactive dashboard using charts and slicers
- Analyzed patterns based on income, age group, and region

 What I found
- Customers with higher income are more likely to purchase bikes
- Certain age groups show stronger buying patterns
- Sales vary across different regions
